============== 후기 ===============

jsp와 js를 나눔으로써 서버 동작 순서가
JAVA -> JSTL -> HTML -> Javascript 인것을 뼈저리게 느꼈고, 비동기 처리 후 출력 시, 애를 많이 먹었다.
하지만 백엔드의 원하는 데이터를 js에서 사용하고 싶을 때, 나름의 여러 방법을 시도해서 해결을 했던 좋은 경험이었다.
또한, 스프링 프레임워크로 프로젝트를 진행하면서 백엔드와 프론트엔드간의 데이터를 주고 받는 방법이나,
학원에서 배웠지만 시도해 볼 기회가 없었던 여러 기능들을 사용해 볼 수 있는 값진 경험이 됐다.
